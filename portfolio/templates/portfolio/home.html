Real-time chat app

'Chat App' is very self-explanatory. The project is a django web application allowing users to chat with themselves in created rooms.

Core features:
User management system - registering, logging, resetting passwords
Friends system - searching, inviting, filtering, removing
Creating rooms and real-time chatting based on Ajax requests

Created with:
Django, AWS RDS PostgreSQL, AWS S3 Bucket, Sass, jQuery, Ajax
